# Challenge Conversor de Moedas.
Desenvolver um conversor de moedas que ofereça interação textual (via console) com os usuários, proporcionando no mínimo 6 opções distintas de conversões de moedas em um menu. Seu principal objetivo é fazer uma chamada à API do site
e converter o valor pré-determinado no menu de conversão; além disso, é possível visualizar um histórico de conversões.
### Tecnologias usadas:
✔️ IntelliJ IDEA<br>
✔️ Java 21<br>
✔️ Biblioteca "Gson" configurada no ambiente de desenvolvimento<br>
✔️ Classes e a interface: HttpCliente, HttpRequest e HttpResponse<br>
✔️ API Key do **Exchangerate-api**

## Como usar:

### Instalação & Configuração:
```bash
git clone https://github.com/seu-usuario/ConversorDeMoedas.git
cd ConversorDeMoedas
```
### ▶️ Como executar o projeto
**No IntelliJ IDEA:**

1) Abra o IntelliJ
2) File → Open → selecione a pasta do projeto
3) Certifique-se de que a biblioteca Gson está adicionada
4) Execute a classe Principal.java
